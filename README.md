Sunshine can be awkward to use with multiple displays setup. I created a virtual display but when I went to sethe the launch arguments to ```sh -c "kscreen-doctor output.DP-1.enable``` it wouldn't actually use the command because sunshine failed to bond to any screen. To get around this I have written a small script to disable one screen that will never be used if I'm streaming to my tablet, enable a virtual screen that is the exact dimensions of my tablet screen and then start sunshine. After receiving an exit code 0 from sunshine it then runs ```kscreen-doctor output.DP-1.disable``` to disable the virtual screen so that it isn't a bother when using the pc.
